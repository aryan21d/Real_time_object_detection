# 🎯 Real-Time Object Detection using YOLO
This project implements a real-time object detection system using YOLO (You Only Look Once) and OpenCV. The system captures live video from a webcam and detects multiple objects such as people, vehicles, and everyday items with bounding boxes and confidence scores.


## 🧠 Features
- 🎥 Real-time detection using webcam
- 📦 Multiple object detection (COCO dataset)
- 📊 Bounding boxes with labels and confidence scores
- ⚡ Fast and efficient detection using YOLO
- 🔄 Non-Max Suppression to remove duplicate detections


## 🛠️ Technologies Used
- Python 🐍
- OpenCV
- YOLO (You Only Look Once)
- NumPy
- Streamlit (for web app deployment)
